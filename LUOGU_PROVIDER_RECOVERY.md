# 洛谷训练数据提供方恢复说明

状态：`awaiting_official_permission`（停用）

Leather 当前不提供洛谷账号绑定、验证、提交同步、热力图筛选或推荐来源。原因是洛谷没有提供已获授权的公开提交记录 API，并且公开规则排除了对提交记录页面的自动抓取。历史洛谷绑定仅以 `disabled` 状态保留兼容行；历史提交事件及其派生统计已从活动画像中清除。

## 恢复前置条件

只有同时满足以下条件，才可通过新的迁移和适配器版本恢复：

1. 获得洛谷书面许可，或洛谷发布覆盖所需数据的官方 API/OAuth。
2. 书面明确允许读取的数据范围、调用频率、数据保留期限和用户撤销流程。
3. 新适配器不接收用户密码、Cookie、会话或其他登录凭据。
4. 使用官方测试环境或固定响应样例完成分页、限流、撤销、结构变化和幂等契约测试。
5. 完成安全复核，确认没有任意 URL 抓取、登录代理、验证码绕过或源码采集。
6. 发布新的功能开关迁移，重新验证旧绑定；不得自动恢复旧绑定和旧提交贡献。

## 保留内容

- 数据库中的 `platform='luogu'` 兼容字段、主外键及标签映射。
- `supabase/functions/_disabled/training-luogu-provider.ts` 中不联网的难度映射与恢复契约。
- 停用账号行，仅供未来重新验证和数据迁移参考，不由公共 DTO 返回。

## 明确禁止

- 抓取被排除的提交记录页面。
- 收集或代理洛谷密码、Cookie、登录会话。
- 使用非官方接口冒充官方稳定能力。
- 在没有重新验证用户所有权的情况下恢复历史绑定。
